# Fraction Class Using OOP in Python 

This repository contains a custom implementation of a **Fraction Class using Object-Oriented Programming (OOP) in Python**.  
The project demonstrates how mathematical fraction operations can be implemented using operator overloading.



##  Features Implemented

- Fraction Object Creation
- String Representation of Fractions
- Addition of Fractions
- Subtraction of Fractions
- Multiplication of Fractions
- Division of Fractions
- Operator Overloading using Magic Methods



##  Topics Covered

- Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors (`__init__`)
- Magic Methods
- Operator Overloading
- Mathematical Computation in Python
- Custom Data Types



## 🛠 Technologies Used

- Python
- Jupyter Notebook



##  Goal of This Repository

The main goal of this repository is to understand how Python classes and operator overloading work internally by building a custom Fraction class from scratch.

This project focuses on:
- OOP fundamentals
- Python magic methods
- Custom arithmetic operations
- Problem-solving using classes
